# Face Identifier - JavaScript

A complete face detection and recognition library for JavaScript, inspired by the Python version. Runs entirely in the browser with no server required.

## Features

✅ **Face Detection** - Detect multiple faces in images/video  
✅ **Face Recognition** - Recognize known faces with 512-dim embeddings  
✅ **Face Hashing** - Generate binary hashes for quick searching  
✅ **Local Database** - Store users in localStorage with face images  
✅ **Export/Import** - Save and load data as JSON files  
✅ **No Downloads** - Models load automatically from CDN  
✅ **Works Offline** - Caches models after first load  
✅ **Clean API** - Same interface as Python version  

## Quick Start

### 1. Include the library

```html
<!-- Include face-api.js (required) -->
<script src="https://cdn.jsdelivr.net/npm/face-api.js@0.22.2/dist/face-api.min.js"></script>

<!-- Include Face Identifier -->
<script src="face-identifier.js"></script>
